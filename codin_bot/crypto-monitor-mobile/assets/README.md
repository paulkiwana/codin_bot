# Assets Folder

This folder contains the app icons and splash screens. You'll need to add the following files:

- `icon.png` - App icon (1024x1024 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `splash.png` - Splash screen (1284x2778 px)
- `favicon.png` - Web favicon (48x48 px)
- `notification-icon.png` - Notification icon (96x96 px)

## Quick Setup

You can generate these assets automatically using:

```bash
npx expo install expo-asset
```

Or create simple placeholder images using any image editor. The app will work without custom icons, but will use default Expo icons.

## Recommended Tools

- [Figma](https://www.figma.com/) - Design your icons
- [App Icon Generator](https://www.appicon.co/) - Generate all sizes
- [Expo Asset Generator](https://github.com/expo/expo/tree/main/packages/expo-asset) - Official tool
