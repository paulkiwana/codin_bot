## üì± Quick Reference Cards

### Essential Commands (Windows)

```batch
# Setup
setup.bat              # Install dependencies
test_setup.bat         # Test installation

# Run
start_gui.bat          # Launch GUI
start_monitor.bat      # Launch console

# Manual
python crypto_monitor_gui.py      # GUI
python crypto_monitor.py          # Console
python test_monitor.py            # Tests
```

### Essential Files

```
.env                   # Your configuration (create from .env.example)
crypto_monitor.log     # Activity log
requirements.txt       # Dependencies
```

### Essential Settings

```env
CRYPTO_SYMBOLS=BTC/USDT,ETH/USDT    # What to monitor
CHECK_INTERVAL=300                   # How often (seconds)
RSI_OVERBOUGHT=90                    # Oversold threshold
RSI_OVERSOLD=10                      # Overbought threshold
SR_THRESHOLD=0.02                    # S/R proximity (2%)
```
